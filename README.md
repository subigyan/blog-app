# Blog App

An Blog App built using MongoDB, Express, React and Node.

## Install Api Dependencies

### `cd api`

### `npm install`

## Install React Dependencies

### `cd client`

### `npm install`

### `cd ..`

## Run App

### `cd api`

### `npm run dev`

### Run front end (react) only

### `npm run client`

## Build

### `cd client && npm run build`
